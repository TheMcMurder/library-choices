  <files>package.json, eleventy.config.js, postcss.config.mjs, .gitignore, src/css/style.css, src/_data/config.js</files>

  <action>
1. Run `npm init -y` to create package.json in the project root.

2. Install dev dependencies:
   ```
   npm install --save-dev @11ty/eleventy tailwindcss @tailwindcss/postcss postcss npm-run-all2
   ```

3. Update package.json scripts to exactly:
   ```json
   {
     "scripts": {
       "start": "npm-run-all -p dev:*",
       "build": "run-s build:css build:11ty",
       "build:11ty": "eleventy",
       "build:css": "tailwindcss -i src/css/style.css -o src/css/compiled.css",
       "dev:11ty": "eleventy --serve",
       "dev:css": "tailwindcss -i src/css/style.css -o src/css/compiled.css --watch"
     }
   }
   ```
   Also add `"engines": { "node": ">=18" }` to package.json.

4. Create `eleventy.config.js` (ESM format) with this exact content:
   ```javascript
   import { HtmlBasePlugin } from "@11ty/eleventy";

   export default function (eleventyConfig) {
     eleventyConfig.addPlugin(HtmlBasePlugin);
     eleventyConfig.addPassthroughCopy("src/css");
     eleventyConfig.addPassthroughCopy("src/js");
     eleventyConfig.addWatchTarget("src/css/");

     return {
       dir: {
         input: "src",
         output: "_site",
       },
       pathPrefix: "/library-choices/",
     };
   }
   ```

5. Create `postcss.config.mjs` with this exact content:
   ```javascript
   export default {
     plugins: {
       "@tailwindcss/postcss": {},
     },
   };
   ```

6. Create `.gitignore` with these entries:
   ```
   _site/
   node_modules/
   .cache/
   *.log
   src/css/compiled.css
   ```
   Note: `src/css/compiled.css` is a build artifact (Tailwind output) and must not be committed.

7. Create directory `src/css/` and file `src/css/style.css` with:
   ```css
   @import "tailwindcss";
   ```
   No `@tailwind base/components/utilities` — that is v3 syntax. No `tailwind.config.js` file.

8. Create directory `src/_data/` and file `src/_data/config.js` with:
   ```javascript
   module.exports = {
     siteName: "Cache County Library Choices",
   };
   ```
   This verifies the Eleventy data cascade works. The `config` object will be available in templates as `{{ config.siteName }}`.

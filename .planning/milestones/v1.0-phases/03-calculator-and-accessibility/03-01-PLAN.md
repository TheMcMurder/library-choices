4. Create `updateResult()` function:
   - Call `getTotalHouseholds()` first
   - Zero-city guard: `if (totalHouseholds === 0)` — set `resultEl.innerHTML` to:
     ```
     '<h2 class="text-xl font-semibold text-gray-900">Annual cost per household</h2>' +
     '<p class="text-sm text-gray-600 mt-2">Select at least one participating city to calculate the estimated cost per household.</p>'
     ```
     Then `return` immediately — never divide.
   - Calculate: `const totalCost = getStaffingCost() + getCollectionsCost();`
   - Calculate: `const perHousehold = totalCost / totalHouseholds;`
   - Set `resultEl.innerHTML` to:
     ```
     '<h2 class="text-xl font-semibold text-gray-900">Annual cost per household</h2>' +
     '<p class="text-3xl font-semibold text-blue-700 mt-2">$' + perHousehold.toFixed(2) + '</p>' +
     '<p class="text-sm text-gray-500 mt-1">$' + totalCost.toLocaleString('en-US') + ' total ÷ ' + totalHouseholds.toLocaleString('en-US') + ' households</p>'
     ```

5. Attach a single delegated `change` event listener on `form`: `form.addEventListener('change', updateResult);`

6. Call `updateResult()` immediately at the bottom of the IIFE (script is at end of body, DOM is ready — no DOMContentLoaded wrapper needed).
